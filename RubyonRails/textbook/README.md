# textbook

## コマンド
プロジェクト作成
```
rails new ProjectName
```

ローカルホストの立ち上げ
```
rails server
```

コントローラー作成
```
rails generate controller ContorollerName
```

モデル作成<br>
ModelNameでモデルの名前を設定し、そのあとでデータの名前とそのデータの型を設定する。
```
rails generate model ModelName data1:dataformat1 data2:dataformat2
```

マイグレーション<br>
テーブルが作成される
```
rails db:migrate
```
